# TaxTim Crypto Tax Calculator

A South African cryptocurrency tax calculator built for TaxTim.

This application helps users calculate capital gains and losses on cryptocurrency transactions using the FIFO (First-In, First-Out) method, in line with SARS requirements.

## Features

- FIFO-based capital gains calculation
- Support for buy and sell transactions
- Clear summary of gains and losses
- Built with modern web technologies
- Fast and responsive UI

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
